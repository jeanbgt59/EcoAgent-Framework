Commandes Principales

ecoagent create

Terminal
$ ecoagent create "mon-app"
$ ecoagent create "e-commerce" --template store
$ ecoagent create "blog" --mode lightweight
$ ecoagent create "api" --type backend-only

Génère une application complète avec FastAPI + React + PostgreSQL + Docker

ecoagent demo

Terminal
$ ecoagent demo ecommerce
$ ecoagent demo blog --lang fr
$ ecoagent demo crm --industry tech
$ ecoagent demo portfolio --style modern

Lance des démonstrations prêtes à l'emploi pour différents secteurs

ecoagent status

Terminal
$ ecoagent status
$ ecoagent system --check
$ ecoagent costs --month
$ ecoagent agents --health

Affiche l'état du système, coûts, performances et santé des agents

ecoagent config

Terminal
$ ecoagent config --mode standard
$ ecoagent config --lang fr
$ ecoagent config --budget-max 5
$ ecoagent config --local-only true

Configure les préférences, modes d'exécution et limitations

Gestion d'Erreurs et Validation

Prévention Proactive

RAM Insuffisante

Votre système dispose de 6GB RAM. Le mode léger nécessite 8GB minimum.

Suggestion: Fermez les applications inutiles ou utilisez un VPS.

Budget Dépassé

Cette génération coûtera ~7€, votre limite est de 5€.

Suggestion: Utilisez --local-only ou augmentez votre budget.

Messages d'Erreur Clairs

❌ Erreur de Configuration
│
├─ Problème: Template 'ultra-modern' introuvable
├─ Templates disponibles:
│ • basic, modern, professional
│ • ecommerce, blog, dashboard
│
💡 Solution suggérée:
ecoagent create "mon-app" --template modern
